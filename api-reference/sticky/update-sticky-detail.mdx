---
title: Update a sticky
description: Updates an existing sticky by setting the values of the parameters passed. Any parameters not provided will be left unchanged.
api: PATCH /api/v1/workspaces/{workspace_slug}/stickies/{sticky_id}/
---

## Path parameters

<ParamField path="workspace_slug" type="string" required>
The workspace_slug represents the unique workspace identifier for a workspace in Plane. It can be found in the URL. For example, in the URL `https://app.plane.so/my-team/projects/`, the workspace slug is `my-team`.
</ParamField>

<ParamField path="sticky_id" type="string" required>
The unique identifier for the sticky.
</ParamField>

## Body parameters

<ParamField body="name" type="string">
Name of the sticky.
</ParamField>

<ParamField body="description_html" type="string">
HTML-formatted content of the sticky.
</ParamField>

<ParamField body="logo_props" type="object">
Logo properties and visual configuration.
</ParamField>

<ParamField body="color" type="string">
Text color for the sticky (hex code).
</ParamField>

<ParamField body="background_color" type="string">
Background color for the sticky (hex code). 
</ParamField>
